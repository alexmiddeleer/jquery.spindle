General Usage

Add jquery.spindle.js and jquery.spindle.css to your page.

Spindle spinners can be added anywhere via: $('...').spindle();. You can also...

  • Explicitly start a spinner:
    $('...').spindle('start');
  • Explicitly stop a spinner:
    $('...').spindle('stop');
  • Toggle a spinner's state:
    $('...').spindle(); /* you can pass 'toggle' too */
  • Provide your own spinner text (js):
    $('...').spindle({text: '@'});
  • Provide your own spinner text (html):
    <span data-spindle-text="@"></span>
  • Add custom css to a spinner:
    $('...').spindle({css: {paddingRight: '10px'}});
  • Change global defaults:
    $.fn.spindle.defaults.text = '$';
    There's also a default css property to add 5px of right padding.
